Projects

Vim Clone

The Fall term of 2018 was my last chance to get my Computer Science average up if I was to have any chance of transferring into my program of choice. As a result, I busted my rear-end in an effort to maximize my grades in the two Computer Science courses I took that term. This project is the result of many hours of toiling away at the keyboard. After the weeks of staying up late writing classes and fixing bugs, my teammate and I were successful in creating vm, a clone of the popular text editor, Vim.

This clone implements a subset of Vim's functionality, including moving the cursor, insert mode, replace mode, yank, delete, and paste, infinite undos, recordable macros, and many more!

Ship Attack Game

In grade 10, I took the Computer Science elective offered at my high school. This is where I truly fell in love with programming. We spent the year learning Object-Oriented Java programming and ended with an open final project. The task: Make a game. This project is what came out of that undertaking. The visuals may not be terribly impressive, but to this day, I am proud of this game I built from scratch.

The game is a simple top-down shooter. Using your blaster, fire at enemy ships and survive the oncoming onslaught as long as possible. It scrolls infintely, but once you accumulate enough points, the waves of enemies get tougher and the tension rises.

Maze Solver

During one of my winter breaks, I decided that I would should learn a new programming language. If I've learned anything from that Udemy guy they show before every YouTube video, it's that “Python’s where it's at.” So, once I learned some of the basic syntax, I decided to write an implementation of the A* pathfinding algorithm to start getting used to the language. This would kill two birds with one stone as I could learn about a prominent pathfinding algorithm while also working on my Python skills.

Once I had finished my maze solving script, I found testing it to be quite tedious. In order to make testing smoother and to further develop my Python, I also made a GUI to accompany my Solver script. It is made using the Python turtle, and allows the user to design mazes and then immadiately see if they are solvable.
